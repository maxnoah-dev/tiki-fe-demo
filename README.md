# Tiki FE Demo - React E-commerce App

Dự án demo frontend React mô phỏng Tiki để học cách deploy lên server.

## Tính năng

- Danh sách sản phẩm với hình ảnh và giá
- Giỏ hàng với thêm/xóa/cập nhật số lượng
- Responsive design (mobile-friendly)
- UI/UX hiện đại với animations
- Sẵn sàng deploy với Docker

## Cài đặt và chạy local

### Yêu cầu
- Node.js >= 18
- npm hoặc yarn

### Cài đặt

```bash
# Cài đặt dependencies
npm install
```

### Chạy development server

```bash
npm run dev
```

Ứng dụng sẽ chạy tại `http://localhost:3000`

### Build production

```bash
npm run build
```

Files sẽ được build vào thư mục `dist/`
